Yet another Project Website!

June 8, 2022 by Prabal


Hey everyone! I bring forth to you yet another project, which I've put my heart and soul into, cause, trust me, this project bolstered me to disenthrall my mind from some regular development stuff and become aquainted with new frameworks and back-end technologies out there.

Thus, I've tried to make a fully responsive website with the help of Bootstrap framework, and parts of which contain SCSS, Tailwind CSS, Less, Vue JS and others. My earlier project was mainly focused on using HTML, CSS and Javascript, so I decided to learn and make use of different frameworks as well from now.
This project is still an ongoing project as I want to continue and make this a full-stack website. Though, I'm already familiar with PHP language and DBMS like MySQL and SQL Server, I've not yet added these so as to contribute as the back-end and database of the website, respectively, owing to some minor system faults.

Since this is deemed to-be a full-stack website, I have had the following technologies and frameworks in mind to work upon. Feel free to reach out to me and recommend me any stuff related to this, it would mean a lot!

  • Since most of the front-end is already developed, I'd just be focusing upon tidying things up, using other CSS and JS frameworks.
  • I'd be learning the major part and implementation of ReactJS as well so that I could create more of SPAs rather than multi-page applications.
  • For creating a robust backend architecture to fetch data from servers, I'd be implementing Python with Django framework or otherwise PHP, which I've already worked upon.
  • For database and servers resilient to outrages, DBMS like MySQL or SQL Server.
  • Version Control to manage project files and keep track of the history, Git, Github and subversion.
  • Cross platform compatibility and responsiveness already ensured by using Bootstrap framework.
  • Web hosting platforms, such as Netlify or Firebase or AWS.

This website shows the user the latest ongoing news in the web-series, TV Shows and Anime/Shounen world.

I have duly taken my references from a number of real-time websites around the world. Some of which are listed:

  1. Screenrant
  2. Fandom
  3. IMDB
  4. Den of Geek
These websites helped me for making some new designs and think upon some creative ideas, in which the website consists of most of the components in the given websites and even some new merger of them, alongwith new innovative utilities.

Idea:

I have always been mesmerized by the Web series world and how it engrasps the viewer and impels them to fly to a whole new space, full of imagination, wonder and fantasy. Thus, it was deemed for me to take upon this hobby o' mine and try to collaborate it with my passion and give upon something which actually counts.


About the Website:

This website, for now, gives the user of the latest news and reportage about numerous TV-Shows and web series, covering almost all genres which exists:

  • Action.
  • Sit-Coms.
  • Muder-Mystery.
  • War.
  • Anime/Shounen World.
  • Drama.
  • Romance.
  • Sci-Fi.
  • Fanstasy and many others.

The future plans for this website is to make this a community, where all the members of our community could interact with each other, have dicussions, forums and rant about their favourite TV-Shows.
